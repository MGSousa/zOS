# zOS 

Install a set of custom dot files

## Prerequisites
 - GNU distribution [debian / arch] with Bash
 - curl

## Install
```sh
git clone https://github.com/MGSousa/zOS && cd zOS && rm -f README.md && chmod +x install
```
